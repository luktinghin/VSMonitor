VSDisplay

VSDisplay is a web viewer for data visualization of VSCapture .CSV files.

You can drop-and-drop .CSV files generated from VSCapture to begin. Then, by creating a row of parameter display, you can visualize the data using a trend line on a chart and an information readout. Basic customization options are provided via "options". Limited functionality now as VSdisplay is currently being actively developed.
